# Projeto-Pedidos
Projeto universitário. Projeto de controle de pedidos feito em JAVA com o intuito de utilizar técnicas da orientação a objetos.

Este projeto possibilita ao usuário:

- Adicionar Clientes
- Adicionar Vendedores
- Adicionar Produtos
- Emitir Pedido
- Calcular a comissão de um vendedor

Também é utilizado métodos de validação de Códigos e CPF.
